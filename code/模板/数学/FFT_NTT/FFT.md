# FFT
## 复数与单位根
复数的指数形式${a+bi=re^{i\theta}}$其中${r=\sqrt{a^2+b^2},tan\theta=\frac{b}{a}}$
## 单位根
${x^n=1}$在复数域上的根称为n次方单位根。n次方单位根有n个，形式为${\omega^k_n=e^{i\frac{2k\pi}{n}}}$
## DFT
${A(\omega^k_n)=B(\omega^k_\frac{n}{2})+\omega^k_nC(\omega^k_\frac{n}{2})}$

${A(\omega^{k+\frac{n}{2}}_n)=B(\omega^k_\frac{n}{2})-\omega^k_nC(\omega^k_\frac{n}{2})}$
